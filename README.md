# Word-Blogger
Ethereum Blockchain backed blogging site, which runs on smart contract made using solidity.

## Packages and Tools needed :
1. Truffle
2. Ganache
3. Node.js
4. npm
5. Metakmask browser extension

## Commands :
```
npm install
truffle test
truffle compile
truffle migrate
npm run start
```
